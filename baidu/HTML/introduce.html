简介 


    Vue.js是一套构建用户界面的渐进式框架。与其他重量级框架不同的是，Vue 采用自底向上增量开发的设计。Vue 的核心库

只关注视图层，并且非常容易学习，非常容易与其它库或已有项目整合。另一方面，Vue 完全有能力驱动采用单文件组件和Vue

生态系统支持的库开发的复杂单页应用。


    Vue.js的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件 
[2]
  。


    Vue.js自身不是一个全能框架——它只聚焦于视图层。因此它非常容易学习，非常容易与其它库或已有项目整合。另一方

面，在与相关工具和支持库一起使用时
[3]
  ，Vue.js也能完美地驱动复杂的单页应用。
特点 

                
    易用

                
    在有HTML CSS JavaScript的基础上，快速上手。

                
    灵活

                
    简单小巧的核心，渐进式技术栈，足以应付任何规模的应用。

                
    性能

                
    20kb min+gzip 运行大小、超快虚拟 DOM 、最省心的优化。
